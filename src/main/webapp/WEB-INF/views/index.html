RUTAS Y FUNCIONALIDADES

Comandos Descripción
/list Devuelve todas las palabras que hayan en la base de datos.
/search/{toSearcgWord} Devuelve los datos de la palabra buscada.
/beginsWith/{letter} Devuelve todas las palabras de la base de datos que empiecen por esta letra.
/endsWith/{letter} Devuelve todas las palabras de la base de datos que terminen por esta letra.
/length/{wordLength} Devuelve todas las palabras de la base de datos que tengan este número de letras.
/length/{wordLength}/range/{numberOfWords} Devuelve la cantidad de palabras especificada en la url que tengan la misma cantidad de letras.
/length/{wordLength}/range/{numberOfWords} Devuelve la cantidad de palabras especificada en la url que tengan la misma cantidad de letras.
/beginsWith/{firstLetter}/range/{numberOfWords} Devuelve la cantidad de palabras especificada en la url que empiecen con la letra de {firstLetter}.
/endsWith/{lastLetter}/range/{numberOfWords} Devuelve la cantidad de palabras especificada en la url que terminen con la letra de {lastLetter}.
